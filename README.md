# order-tracking

I retired in Puerto Rico. I went to the local Post Office to inquire about my
mailing and delivery address. I was informed that there were no mailboxes
available for my address, so I had to get a PO Box. That was fine for  the most part but
when receiving packages, they are supposed to leave a notice in my mailbox, which
usually works, but occasionally they don't. So I wrote a fastapi app to track my
packages for those times I don't receive a notice.

I wrote client app that will check my gmail order notices and update the database.

The client supports amazon and walmart orders.

I am not sure if I will be able to publish the client code, but if you are interested
in the client code, please let me know.
